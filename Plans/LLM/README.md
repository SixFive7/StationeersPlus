# LLM (stub)

The source for this mod is currently held outside the repository. Its full working tree exceeds 2 GB (LLaMA.cpp native runtimes plus model weight files) and does not fit cleanly into a plain git repository.

Once Git LFS is enabled for `SixFive7/StationeersPlus` (see the root `TODO.md`), the source will be committed here with model weight files (`*.gguf`) and native runtime binaries either excluded via `.gitignore` or tracked via LFS, as appropriate.

Until then, `Plans/LLM/*` is gitignored at the repository root except for this README.
